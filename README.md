# FairyGUI Unity

FairyGUI Unity SDK v5.2.0，打包为 UPM 格式。

## 安装

在 Unity 项目的 `Packages/manifest.json` 中添加：

```json
{
  "dependencies": {
    "com.veryfreestyle.unity.fairygui": "https://github.com/veryfreestyle/fairygui-upm.git"
  }
}
```

指定 commit 或 tag：

```json
"com.veryfreestyle.unity.fairygui": "https://github.com/veryfreestyle/fairygui-upm.git#v5.2.0"
```

## 要求

- Unity 2021.3+
- TextMeshPro
